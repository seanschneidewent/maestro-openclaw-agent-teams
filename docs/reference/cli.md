# CLI Reference

## Core Commands

- `maestro-setup`
- `maestro update`
- `maestro doctor --fix`
- `maestro up`
- `maestro serve`
- `maestro-purchase`
- `maestro ingest`
- `maestro index`
- `maestro tools`

## Recommended End-User Path

1. setup once: `maestro-setup`
2. run daily: `maestro up`
3. when upgrading: `maestro update`

## Project Maestro Provisioning

`maestro-purchase` provisions a dedicated project agent and returns ingest guidance.

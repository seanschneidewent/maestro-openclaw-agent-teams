# Glossary

- **Commander**: default company control-plane agent (`maestro-company`)
- **Project Maestro**: project-specific agent (`maestro-project-*`)
- **Fleet Store**: root directory containing project stores
- **Command Center**: control-plane frontend at `/command-center`
- **Workspace Frontend**: per-project UI at `/{slug}` or `/agents/{agent_id}/workspace`
- **Heartbeat**: optional per-project status overlay file
- **Directive**: persisted system alignment record
